# reps
 
## Projeto
* Projeto de compartilhamento de repositórios e API's.
* Foi criado um login em memória e em Banco de Dados.

## Desenvolvimento
* Foi usado Java 1.8.
* Thymeleaf 3.0.12.
* Spring Framework 5.3.7.
* Spring DevTools.
* MariaDB de Banco de Dados.
